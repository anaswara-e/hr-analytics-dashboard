# hr-analytics-dashboard
Interactive HR Analytics Dashboard project using Power BI

Project Overview

The HR Analytics Dashboard is an interactive data visualization project designed to provide actionable insights into workforce trends. Built using Power BI, Power Query, and DAX, the dashboard helps HR teams analyze employee attrition, salary distribution, age demographics, and education levels to make data-driven decisions.

Key Features

Employee Attrition Analysis: Track attrition trends over time and identify high-risk groups.
Salary Distribution Insights: Visualize salary ranges across departments and roles to support compensation planning.
Demographic Analysis: Examine employee age and education trends to guide talent management strategies.
Interactive Filters & Drill-downs: Explore data dynamically with slicers, filters, and hierarchical drill-downs.
Actionable Insights: Identify factors affecting employee retention and workforce performance.

Tools & Technologies

Power BI: Dashboard creation and visualization.
Power Query: Data cleaning, transformation, and preparation.
DAX (Data Analysis Expressions): Measures and calculations for advanced analytics.
Excel (optional): Initial data import and pre-processing.

Data Sources

Employee HR datasets including:

Employee demographics (age, education, gender)
Salary information
Attrition status and history
Department and role details

Project Highlights

Applied data cleaning and transformation techniques to prepare HR data for analysis.
Created dynamic visualizations to enable real-time insights for HR decision-making.
Developed DAX measures to calculate attrition rates, average salaries, and other KPIs.
Provided recommendations based on trends, such as retention strategies and salary adjustments.

How to Use

Open the .pbix file in Power BI Desktop.
Use slicers and filters to explore data by department, role, age, or education.
Drill down into charts for detailed insights at the individual employee or team level.
Export reports or dashboards as PDF or Power BI Service for sharing with stakeholders.

Outcome / Impact

Enabled the HR team to identify key drivers of employee attrition.
Assisted in salary benchmarking and workforce planning.
Supported strategic decision-making through data-driven insights.
